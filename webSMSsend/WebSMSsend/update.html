Hier kann die entsprechende Version von webSMSsend ausgewählt werden:

  1. Noch Version 0.65.5, in den nächsten Tagen folgt Version 0.70.0:
    signierte Ovi Store-Version (Sicherheitsabfragen deaktivierbar)
  2. Version 0.70.0:
    unsignierte Version (Sicherheitsabfragen nicht deaktivierbar)

Achtung! Eine schon installierte Ovi Store-Version kann nur mit einer Ovi Store-Version aktualisiert werden. Falls man zu einer anderen Version wechseln möchte, muss dazu die Ovi Store-Version von webSMSsend zuvor deinstalliert werden. Dabei gehen alle Einstellungen verloren.
