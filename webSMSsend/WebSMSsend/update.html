Hier kann die entsprechende Version von webSMSsend ausgewählt werden:

  1. signierte Ovi Store-Version (Sicherheitsabfragen deaktivierbar)
  2. unsignierte Version (Sicherheitsabfragen nicht deaktivierbar)

Achtung! Eine schon installierte Ovi Store-Version kann nur mit einer Ovi Store-Version aktualisiert werden. Sollte die Aktualisierung über den obigen Link nicht funktionieren, kann die aktuellste Version auch direkt im Ovi Store heruntergeladen werden. Wird keine "Herunterladen"-Schaltfläche angeboten, muss die bisherige webSMSsend-Installation zuvor vom Handy entfernt werden.
Falls man zu einer unsignierten Version wechseln möchte, muss dazu die Ovi Store-Version von webSMSsend ebenfalls zuvor deinstalliert werden. Dabei gehen alle Einstellungen verloren.
