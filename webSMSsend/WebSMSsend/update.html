Hier kann die entsprechende Version von WebSMSsend ausgewählt werden:

1. signierte OviStore-Version (Sicherheitsabgfragen deaktivierbar)
2. unsignierte Version (Sicherheitsabfragen nicht deaktivierbar)

Achtung! Eine schon installierte OviStore-Version kann nur mit einer OviStore-Version aktualisiert werden. Falls man zu einer anderen Version wechseln möchte, muss dazu die OviStore-Version von WebSMSsend zuvor deinstallieren.
